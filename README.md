# Naughty Bean Consulting

Static GitHub Pages site for Naughty Bean Consulting.

## Structure

- `index.html` home page
- `about/` about page
- `contact/` contact page
- `services/` services overview
- `services/invest-in-a-coffee-shop/` coffee shop investment advisory page
- `services/software-engineering/` software engineering services page
- `privacy/` POPIA-aligned privacy policy
- `terms/` generic terms of service

## Assets

- `assets/css/site.css` shared styles
- `assets/js/site.js` mobile navigation and year handling
- `assets/images/` brand logos and the original Naughty Bean Consulting logo

## Deploy to GitHub Pages

1. Push this repository to GitHub.
2. In GitHub, open `Settings` -> `Pages`.
3. Under `Build and deployment`, choose `Deploy from a branch`.
4. Select your default branch and the `/ (root)` folder.
5. Save, then wait for GitHub Pages to publish the site.

If you later attach a custom domain, add it in the GitHub Pages settings and create the matching DNS records.
